# 绿茵智析 — UI设计需求文档

## 项目概述

**名称**：绿茵智析（Football Analytics）
**类型**：本地部署的足球比赛概率分析系统
**技术栈**：Python 3.13 + Streamlit + DuckDB + Plotly
**访问方式**：http://127.0.0.1:8502

## 当前页面结构

```
侧边栏导航（分组）：
├── 实时
│   ├── ⚽ 今日竞彩 — 比赛卡片、赔率、刷新
│   ├── 📊 价值发现 — EV分析、推荐
│   └── 📋 比赛结果
```

## 核心页面需求

### 1. 今日竞彩（首页）
- 比赛卡片：主队 vs 客队、联赛、时间、状态标签
- 赔率展示：主胜/平局/客胜 三个方框
- 让球赔率行
- 操作按钮：详情、分析
- 筛选：日期、联赛、仅销售中
- 统计：场次、销售中、单关、有模型

### 2. 比赛分析（核心页面）
- 顶部指标：可信度、模型倾向、预期进球、首选比分
- 7个标签页：
  - 核结结论：概率对比图、价值差异表、风险因素
  - 球队状态：Elo、近期战绩、历史交锋
  - 阵容伤停：阵型图、球员列表
  - 模型拆解：各模型概率对比柱状图
  - 比分与进球：热力图、最可能比分、大小球
  - 盘口变化：赔率折线图
  - 数据质量：模型元数据

### 3. 比赛详情
- 完场比分展示
- 近期战绩（W/D/L标签 + 比分）
- 阵型图（绿色球场 + 球员圆形头像）
  - 按位置分行：前锋 → 中场 → 后卫 → 门将
  - 球员：号码圆标 + 姓名 + 位置中文
  - 替补席可展开
- 伤停信息
- 赔率变化折线图

### 4. 价值发现
- 推荐卡片：比赛、联赛、推荐赛果
- 指标：模型概率、市场概率、边际、EV
- 置信度标签：高/中/低

### 5. 过关分析
- 比赛选择器
- 已添加比赛列表（可删除）
- 过关汇总：组合赔率、组合概率、EV、Kelly仓位
- 风险分析：单项EV、破产概率

## 设计要求

1. **企业级视觉**：现代、专业、数据驱动
2. **响应式布局**：适配不同屏幕尺寸
3. **数据可视化**：Plotly图表要清晰易读
4. **交互体验**：流畅、直观、反馈及时
5. **配色一致**：统一的设计语言
