# 绿茵智析

本地优先的足球比赛概率分析系统。数据、数据库、模型和分析结果均保存在 `D:\football-model`，浏览器只负责显示本机 `127.0.0.1` 页面。

## 启动

双击 `run_app.cmd`。启动器会检查服务、必要时后台启动，并自动打开浏览器。

也可以手动执行：

```powershell
D:\football-model\.venv\Scripts\python.exe -m streamlit run D:\football-model\app.py
```

浏览器访问 `http://127.0.0.1:8502`。

如果浏览器没有自动出现，直接在地址栏输入 `http://127.0.0.1:8502`，不要使用 `https://`。

## 当前页面

- 今日竞彩：按比赛卡片展示中国竞彩网官方赛程、销售状态、胜平负SP和让球SP，每60秒刷新并保存赔率快照。
- 比赛分析：从每场比赛的“全面分析”按钮进入，统一组合Dixon–Coles、Poisson状态特征与官方市场先验；结论、EV和比分矩阵使用同一预测结果。
- 分析质量：展示核心结论、球队状态、模型拆解、比分进球、盘口变化和数据质量，并明确列出模型分歧与缺失数据风险。
- 证据明细：展示双方近期比赛、历史交锋、Elo差值、状态差值，以及去水后的官方市场概率变化。
- 单场分析：比分矩阵、胜平负、让球、总进球、SP去水和理论EV。
- 批量分析：通过CSV批量计算比赛概率。
- 历史回测：Log Loss、Brier Score和概率校准。
- 数据中心：本地导入比赛数据，管理未来数据源。
- 模型中心：按联赛训练Dixon–Coles模型、模型注册表和组合模型规划。
- 系统状态：数据库、路径、磁盘和系统分层检查。

## 当前开发数据

- 已导入英超2023/24、2024/25和2025/26三个赛季，共1,140场比赛。
- 已保存3,420条外部市场收盘赔率记录。
- 开发数据来自 `https://www.football-data.co.uk/data.php`。
- 该赔率不是中国体彩官方SP，仅用于建立和验证训练、预测、回测流程。

“今日竞彩”页面使用中国竞彩网实时数据；当本地尚无对应球队训练模型时，快速分析会使用官方SP反推市场先验比分分布，并明确标记为非独立预测。

当前按“当天有竞彩才接入”的原则，只为2026-07-05赛程中的联赛建立模型：

- 瑞典超级联赛：3,465场历史比赛，使用693场时间留出集评估。
- 世界杯：限定2026届48支球队，使用243场国家队比赛训练，使用49场时间留出集评估。

模型使用按时间顺序80/20留出回测，记录Log Loss、Brier Score和赛果命中率。独立模型权重根据留出集表现动态计算，不再使用固定比例。世界杯比赛按中立场处理。

每次打开正式比赛分析都会把模型版本、输入SP、组件概率、最终概率、xG与可信度写入本地predictions表，便于赛后结算和性能监控。XGBoost与神经网络当前标记为实验性，在完成独立校准和逐窗口回测前不会进入生产集成。

当前盘口变化指中国竞彩网官方胜平负SP与让球胜平负SP快照。亚洲盘、大小球、首发伤停与精确场馆天气尚未接入；界面会把这些内容列为缺失数据，不会伪装成已生效特征。
